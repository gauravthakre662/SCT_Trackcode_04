📊 Marketing Campaign Analysis - EDA
📌 Project Overview
This project focuses on Exploratory Data Analysis (EDA) of a marketing campaign dataset to evaluate campaign performance across different marketing channels. The analysis helps identify high-performing channels, measure Return on Investment (ROI), and provide data-driven business recommendations.

🎯 Objective
Analyze marketing campaign performance.
Compare revenue and ROI across channels.
Identify conversion funnel drop-offs.
Generate actionable business insights.
Recommend budget allocation strategies.
